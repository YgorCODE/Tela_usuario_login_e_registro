# Tela Login & Registro PHP

Projeto de login e registro usando PHP, MySQL, HTML, CSS e JavaScript.  
Funciona localmente usando XAMPP.

## Como rodar

1. Copie a pasta do projeto para `C:\xampp\htdocs\`  
2. Abra o XAMPP e inicie Apache e MySQL  
3. Crie o banco de dados `login` e a tabela `users` conforme o projeto  
4. Acesse no navegador: `http://localhost/nomedeseuprojeto/index.php`
